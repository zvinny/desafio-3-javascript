# 🎮 Sobre o Projeto

Este projeto foi desenvolvido como parte dos desafios da plataforma DIO.

O objetivo foi criar uma estrutura de heróis utilizando conceitos fundamentais de programação orientada a objetos com JavaScript.

Durante o desenvolvimento foram utilizados:

- ✅ Classes
- ✅ Objetos
- ✅ Constructors
- ✅ Métodos
- ✅ Estruturas condicionais
- ✅ Variáveis
- ✅ Template Strings

---

# ⚔️ Funcionalidades

O sistema permite criar heróis com:

- Nome
- Idade
- Tipo

E executar ataques diferentes dependendo da classe do personagem.

## 🧙 Tipos disponíveis

| Tipo        | Ataque               |
|-------------|----------------------|
| Mago        | Magia                |
| Guerreiro   | Espada               |
| Monge       | Artes marciais       |
| Ninja       | Shuriken             |

---

# 🧠 Lógica Aplicada

O método `atacar()` identifica automaticamente o tipo do herói e exibe o ataque correspondente.

Exemplo de saída:

```bash
O mago atacou usando magia
```
